Page Slider

A Simple pure JavaScript plugin to make single page websites scroll horizontally on clicks.

Author

Abhijeet Bajracharya http://abhijeetbajracharya.com/

Demo

http://abhijeetbajracharya.com/demo/sliding-pages/

Usage

You will need to include:

The JavaScript file pgSlider.js (or its minified version pgSlider.min.js)

The CSS file pgSlider.css

Issues

Works only if there are four or more pages

The transition is still a bit buggy
